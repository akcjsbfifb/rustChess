# Aprendiendo Rust con The Rust Programming Language

## Objetivo
Leer "The Rust Programming Language" poco a poco, haciendo ejercicios y preguntando dudas.

## Formato de Presentación
- **Contenido completo**: Presentar TODO el contenido de cada capítulo, no resumir
- **Secciones digeribles**: Dividir en partes que se puedan leer y entender antes de hacer preguntas
- **Código incluido**: Incluir todos los listados de código completos
- **Explicaciones detalladas**: Mantener todas las explicaciones del libro original

## Estructura
- Capítulos en: `rust_learning/` (archivos Markdown locales del libro)
- Archivo de progreso: `rust_progress.md`
- Este archivo: `rust_instructions.md`
- Proyectos de práctica: En subdirectorios (ej: `guessing_game/`)

## Flujo de trabajo
1. Presento capítulo/parte del libro con TODO el contenido
2. Espero preguntas o confirmación para continuar
3. Hacemos ejercicios juntos si los hay
4. Actualizo automáticamente `rust_progress.md`
5. Paso al siguiente capítulo/sección
