# Aprendiendo Rust con The Rust Programming Language

## Objetivo
Leer "The Rust Programming Language" en la web oficial y hacer los ejercicios de rustlings, haciendo preguntas cuando surjan dudas.

## Enfoque Actual
- **Libro**: Leer en https://doc.rust-lang.org/stable/book/ (web)
- **Ejercicios**: rustlings (ejercicios prácticos de código)
- **Dudas**: Preguntar al asistente cuando sea necesario

## Ejercicios de Rustlings
- Repositorio: https://github.com/rust-lang/rustlings
- Instalación: `cargo install rustlings`
- Ejecución: `rustlings` (en el directorio de rustlings)
